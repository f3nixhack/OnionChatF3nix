# OnionChatF3nix
OnionChatF3nix es una app de chat P2P sobre Tor (.onion), ligera y con interfaz de escritorio simple. Permite crear salas onion efímeras, conectarse de forma segura por Tor, enviar mensajes en tiempo real y compartir archivos (hasta 5 MB), con detección automática de SOCKS/ControlPort y temas claro/oscuro.
